# GitHub Profile Setup

这个文件夹已经准备好一个 GitHub 个人主页模板。

## 怎么上线

1. 在 GitHub 新建一个仓库，仓库名必须和你的 GitHub 用户名完全一致。
   - 例如用户名是 `octocat`，仓库名就叫 `octocat`。
2. 把这个文件夹里的 `README.md` 和 `assets/profile-banner.svg` 上传到那个仓库。
3. 打开你的 GitHub 个人主页，就会自动显示这份主页。

## 需要替换的内容

在 `README.md` 里搜索并替换：

- `georgezhong10`：如果这不是你的 GitHub 用户名，请替换成真实用户名。

在 `assets/profile-banner.svg` 里搜索并替换：

- `george`：如果想显示中文名或全名，请替换这里。
